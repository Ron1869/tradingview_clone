Полный промт для веб-приложения для трейдинга на React

**Общее описание:**

Современное веб-приложение для трейдинга, построенное на React. Платформа предоставляет пользователям широкий набор инструментов для анализа рынка, торговли, социального взаимодействия и автоматизации. Приложение спроектировано как Single Page Application (SPA) с использованием React Router для навигации.

**Технологический стек:**

*   **Фреймворк:** React 18
*   **Сборщик:** Vite
*   **Управление состоянием:** Redux Toolkit
*   **Стилизация:** TailwindCSS
*   **Маршрутизация:** React Router v6
*   **Визуализация данных:** D3.js, Recharts
*   **Формы:** React Hook Form
*   **Анимация:** Framer Motion
*   **Тестирование:** Jest, React Testing Library

---

### **Структура страниц и компонентов:**

#### 1. **Страница Регистрации (`/registration`)**

*   **Назначение:** Регистрация новых пользователей в системе.
*   **Основные блоки:**
    *   **Логотип и приветствие:** Брендинг и вовлекающее сообщение для новых пользователей.
    *   **Форма регистрации (`RegistrationSteps`):** Многошаговая форма:
        1.  **Шаг 1: Создание аккаунта:** Поля для ввода email и пароля (с подтверждением). Валидация данных в реальном времени (корректность email, надежность пароля).
        2.  **Шаг 2: Личная информация:** Поля для имени, фамилии, страны (опционально).
        3.  **Шаг 3: Верификация:** Инструкции по верификации аккаунта (например, через email).
    *   **Кнопки социального входа (`SocialLoginButtons`):** Интеграция с Google, Facebook, Apple для быстрой регистрации без пароля.
    *   **Выбор языка (`LanguageSelector`):** Выпадающий список или иконки флагов для выбора языка интерфейса на начальном этапе.
    *   **Ссылка на страницу входа:** Для пользователей, у которых уже есть аккаунт.

#### 2. **Страница Входа (`/login`)**

*   **Назначение:** Аутентификация существующих пользователей.
*   **Основные блоки:**
    *   **Форма входа:** Стандартные поля "Email" и "Пароль".
    *   **Кнопки социального входа (`SocialLoginButtons`):** Быстрый вход через социальные сети.
    *   **Ссылка "Забыли пароль?":** Перенаправляет на флоу восстановления доступа к аккаунту.
    *   **Опция "Запомнить меня":** Чекбокс для сохранения сессии пользователя.
    *   **Ссылка на страницу регистрации:** Для новых пользователей.

#### 3. **Главная Панель / Дашборд (`/dashboard`)**

*   **Назначение:** Центральная страница, предоставляющая пользователю сводную информацию о состоянии его портфеля и рынка в целом.
*   **Основные блоки:**
    *   **Карточки с ключевыми метриками (`MetricsCards`):** Серия карточек в верхней части страницы для быстрого обзора:
        *   Общая стоимость портфеля.
        *   Изменение за 24 часа (в % и в валюте).
        *   Доступный баланс для торговли.
        *   Открытые позиции (количество).
    *   **Сводка по портфелю (`PortfolioSummary`):**
        *   **Визуализация:** Круговая диаграмма (Pie Chart), показывающая распределение активов в портфеле.
        *   **Таблица:** Список активов с указанием количества, текущей цены, общей стоимости и суточного изменения.
    *   **Список наблюдения (`MarketWatchlist`):** Настраиваемый список отслеживаемых пользователем активов. Колонки: Тикер, Название, Цена, Изменение (%), График-миниатюра (sparkline).
    *   **Быстрый торговый график (`TradingChart`):** Интерактивный, но упрощенный график для актива, выбранного из списка наблюдения или портфеля.
    *   **Панель быстрой торговли (`TradingPanel`):** Упрощенная форма для мгновенного исполнения рыночных ордеров (покупка/продажа).
    *   **Лента рыночных новостей (`MarketNews`):** Агрегатор новостей из различных источников, релевантных для активов пользователя.
    *   **Недавние сделки (`RecentTrades`):** Таблица с историей последних исполненных ордеров пользователя.

#### 4. **Расширенный График (`/advanced-chart`)**

*   **Назначение:** Предоставление профессиональных инструментов для глубокого технического анализа.
*   **Основные блоки:**
    *   **Контейнер с графиком (`ChartContainer`):** Главный элемент страницы. Отображает ценовой график (свечи, бары, линия). Поддерживает масштабирование, панорамирование.
    *   **Панель управления графиком (`ChartControls`):**
        *   **Таймфреймы:** Кнопки для переключения (1m, 5m, 15m, 1h, 4h, 1D, 1W, 1M).
        *   **Индикаторы:** Меню для добавления и настройки технических индикаторов (Moving Averages, RSI, MACD, Bollinger Bands, Volume и т.д.).
        *   **Инструменты рисования:** Панель с инструментами для нанесения на график линий тренда, уровней Фибоначчи, текстовых заметок.
    *   **Боковая панель символов (`SymbolSidebar`):** Поиск и выбор торговых пар/активов. Группировка по категориям (Crypto, Stocks, Forex).
    *   **Глубина рынка (`MarketDepth`):** Визуальное представление книги ордеров (стакана). Показывает объемы на покупку (bid) и продажу (ask) на разных ценовых уровнях.
    *   **Расширенная торговая панель (`TradingPanel`):**
        *   **Типы ордеров:** Market, Limit, Stop-Limit, Stop-Market.
        *   **Поля:** Цена, Количество, Сумма.
        *   **Дополнительные опции:** Take Profit, Stop Loss.
        *   **Расчеты:** Автоматический расчет стоимости и рисков.

#### 5. **ИИ-Помощник (`/ai-assistant`)**

*   **Назначение:** Интерактивный чат с искусственным интеллектом для анализа рынка, генерации идей и ответов на вопросы.
*   **Основные блоки:**
    *   **Интерфейс чата (`ChatMessage`):** Диалоговое окно для общения с ИИ. Приветственное сообщение (`aiWelcomeMessage`).
    *   **Поле ввода запроса:** Текстовое поле с плейсхолдером (`enterPrompt`).
    *   **Быстрые действия (`QuickActions`):** Предустановленные кнопки для частых запросов: "Обзор рынка сегодня", "Проанализируй [символ] и предскажи движение цены", "Какие акции в тренде?".
    *   **Карточки с прогнозами (`MarketPredictionCard`):** Специализированный компонент в ответе ИИ, который структурированно представляет прогноз: символ, текущая цена, прогноз (рост/падение), целевая цена, уверенность прогноза.
    *   **Настройки ИИ (`AISettings`):** Модальное окно или боковая панель для детальной настройки поведения ИИ-помощника.
        *   **Профиль риска (`riskProfile`):** Слайдер или радио-кнопки для определения толерантности пользователя к риску.
            *   *Консервативный:* ИИ предлагает идеи с низким риском, фокусируется на сохранении капитала.
            *   *Умеренный:* Сбалансированный подход между риском и доходностью.
            *   *Агрессивный:* ИИ ищет высокодоходные, но и высокорисковые возможности.
        *   **Стиль торговли (`tradingStyle`):** Выбор предпочитаемого стиля торговли пользователя.
            *   *Дневная торговля (Day Trading):* Анализ и идеи для внутридневных сделок.
            *   *Свинг-трейдинг (Swing Trading):* Фокус на удержании позиций от нескольких дней до недель.
            *   *Скальпинг (Scalping):* Идеи для очень коротких сделок с целью получения небольшой прибыли.
            *   *Долгосрочное инвестирование:* Анализ фундаментальных показателей для долгосрочных вложений.
        *   **Предпочитаемые рынки (`preferredMarkets`):** Чекбоксы для выбора рынков, на которых ИИ должен сфокусировать свой анализ (Криптовалюты, Акции, Forex, Товары).
        *   **Глубина анализа (`reasoningDepth`):** Выпадающий список для выбора уровня детализации ответов.
            *   *Быстрый обзор:* Краткие выводы и основные моменты.
            *   *Стандартный анализ:* Включает ключевые уровни поддержки/сопротивления и основные индикаторы.
            *   *Глубокий анализ:* Детальный разбор с несколькими индикаторами, фундаментальными данными и анализом настроений.
        *   **Источники данных (`dataSources`):** Переключатели для включения/отключения источников, которые ИИ использует для анализа.
            *   Технические индикаторы.
            *   Фундаментальные данные (отчетность, мультипликаторы).
            *   Анализ новостного фона.
            *   Тренды из социальных сетей.
        *   **Язык и тон (`languageAndTone`):** Выбор стиля общения ИИ.
            *   *Профессиональный:* Формальный, технический язык.
            *   *Повседневный:* Более простой и дружелюбный тон.

#### 6. **Управление Оповещениями (`/alerts-management`)**

*   **Назначение:** Создание, просмотр и управление пользовательскими торговыми оповещениями.
*   **Основные блоки:**
    *   **Панель создания оповещений (`AlertCreationPanel`):** Форма для настройки нового оповещения:
        *   Выбор актива (symbol).
        *   Тип условия (цена, технический индикатор, объем).
        *   Условие (выше, ниже, пересекает).
        *   Значение.
        *   Способ уведомления (push-уведомление, email).
    *   **Шаблоны оповещений (`AlertTemplates`):** Галерея готовых шаблонов для популярных сценариев ("Прорыв цены биткоина", "Перепроданность RSI ETH"). Позволяет создать оповещение в один клик.
    *   **Таблица активных оповещений (`AlertsTable`):** Список созданных пользователем оповещений с возможностью их редактирования, приостановки или удаления. Включает индикатор статуса (`AlertStatusIndicator`).
    *   **История оповещений (`AlertHistory`):** Лог всех ранее сработавших оповещений с указанием даты, времени и параметров.

#### 7. **Сканер Рынка (`/market-scanner`)**

*   **Назначение:** Инструмент для поиска активов, соответствующих заданным пользователем критериям.
*   **Основные блоки:**
    *   **Панель фильтров (`FilterPanel`):** Набор разнообразных фильтров:
        *   **Фундаментальные:** Рыночная капитализация, P/E, дивидендная доходность.
        *   **Технические:** % изменения цены (за день, неделю), достижение нового максимума/минимума, значения индикаторов (RSI, MACD).
        *   **Описательные:** Сектор, индустрия, страна.
    *   **Предустановленные сканы (`PresetScans`):** Готовые наборы фильтров, например: "Акции роста", "Недооцененные компании", "Самые волатильные криптовалюты".
    *   **Результаты сканирования (`ScanResults`):** Динамическая таблица, отображающая активы, которые соответствуют заданным фильтрам.
    *   **История сканирования (`ScanHistory`):** Возможность сохранять наборы фильтров и просматривать результаты предыдущих сканирований.

#### 8. **Лента Социальной Торговли (`/social-trading-feed`)**

*   **Назначение:** Социальная платформа для обмена торговыми идеями, аналитикой и мнениями.
*   **Основные блоки:**
    *   **Лента постов:** Бесконечная прокрутка постов от других трейдеров.
    *   **Карточка поста (`PostCard`):**
        *   **Автор:** Аватар, имя, верификационный статус, количество подписчиков.
        *   **Контент:** Текст, прикрепленные изображения или снимки графика.
        *   **Мета-информация:** Хэштеги, временная метка.
        *   **Социальные взаимодействия:** Кнопки "Лайк", "Комментарий", "Поделиться".
    *   **Форма создания поста (`ComposePost`):** Поле для ввода текста, возможность прикрепить изображение или текущий снимок графика со страницы "Расширенный график".
    *   **Панель фильтров (`FilterBar`):** Сортировка ленты: "Подписки", "Популярное", "Последние".
    *   **Боковая панель трендов (`TrendingSidebar`):** Отображение популярных хэштегов, самых обсуждаемых активов и самых влиятельных трейдеров.

#### 9. **Страница не найдена (`/404`)**

*   **Назначение:** Отображается при переходе по несуществующему URL.
*   **Блоки:**
    *   Сообщение об ошибке "404 - Страница не найдена".
    *   Забавная иллюстрация или анимация.
    *   Кнопка/ссылка для возврата на главную страницу.
